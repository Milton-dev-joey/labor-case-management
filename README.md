# 劳动案件管理系统

律师个人劳动案件管理桌面应用，支持案件录入、紧迫度追踪、文书生成。

---

## 下载使用（Windows 用户）

1. 前往 [Releases 页面](../../releases/latest)
2. 下载 `劳动案件管理系统-*-portable.exe` 文件
3. 双击运行，**无需安装**

> **首次运行提示：** Windows SmartScreen 可能弹出蓝色安全提示，点击「更多信息」→「仍要运行」即可正常启动。这是未签名软件的正常现象，不影响使用。

### 系统要求
- Windows 10 / 11（64 位）
- 无需安装任何其他软件

---

## 数据备份

所有案件数据保存在本地，路径为：

```
%APPDATA%\劳动案件管理系统\
```

建议定期将该目录复制备份，防止数据丢失。

---

## 自动更新

应用启动后约 3 秒会自动检查 GitHub Releases 中的新版本：
- 发现新版本时会提示正在后台下载
- 下载完成后点击「立即重启」完成更新

---

## 主要功能

- **案件管理** — 新建、编辑、删除劳动案件，支持批量录入
- **紧迫度指示** — 根据开庭日期自动显示红/黄/绿状态
- **我方主体库** — 管理常用公司信息，创建案件时快速选择
- **办案人管理** — 录入办案律师信息
- **文书生成** — 自动生成答辩状、授权委托书、法定代表人身份证明书并导出 Word
- **搜索筛选** — 按案件名称、案由、当事人搜索

## 发布新版本（开发者）

```bash
# 修改 package.json 中的版本号后：
git add .
git commit -m "chore: bump version to v1.x.x"
git tag v1.x.x
git push origin main
git push origin v1.x.x
```

推送 tag 后 GitHub Actions 会自动构建并发布新的 Release。
